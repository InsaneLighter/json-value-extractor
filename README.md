# JSON 值提取器

一个简单的 VSCode 扩展，用于从 JSON 文件中提取指定字段的所有值并合并。

## 功能

- 从 JSON 文件中提取指定字段的所有值
- 支持排序（升序/降序）
- 支持自定义分隔符
- 显示值的统计信息（出现次数等）
- 支持大文件处理

## 使用方法

1. 在 VSCode 中打开一个 JSON 文件
2. 按下 `Ctrl+Shift+P`（Windows/Linux）或 `Cmd+Shift+P`（Mac）打开命令面板
3. 输入 "提取 JSON 字段值" 并选择该命令
4. 输入要提取的字段名
5. 选择排序方式
6. 输入分隔符（默认为逗号）
7. 结果将在新标签页中显示

## 安装

1. 打开 VSCode
2. 打开扩展面板（Ctrl+Shift+X）
3. 搜索 "JSON 值提取器"
4. 点击安装

## 卸载

1. 打开 VSCode
2. 打开扩展面板（Ctrl+Shift+X）
3. 找到 "JSON 值提取器"
4. 点击卸载
